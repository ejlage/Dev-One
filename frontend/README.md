# Ent'Artes - Escola de Dança

Frontend da aplicação Ent'Artes - Sistema de Gestão de Escola de Dança

## Tecnologias

- React
- TypeScript
- Vite
- TailwindCSS
- Radix UI

## Como executar

```bash
npm install
npm run dev
```

## Configuração

O frontend comunica com a API em http://localhost:3000
